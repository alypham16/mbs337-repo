# Homework 05

This directory is for homework 5 of MBS 337, Reserach Computing in Biology.

## Setup

The exercises for Homework 04 were ran in a virtual environment with the **biopython** package installed:

```bash
source myenv/bin/activate
pip3 install biopython
```

The following file was used for this homework:

- 4HHB.cif (hemoglobin structure): https://files.rcsb.org/download/4HHB.cif.gz

## Exercise Description

This homework requires installation of the **biopython** package.

The exercise includes building a **mmcif_summary.py** Python script htat reads a mmCIF structure file, computes per-chain residue statistics, and writes the result to a JSON file in a specific format (https://mbs-337-sp26.readthedocs.io/en/latest/homework/homework05.html)

## Miscellaneous

- output_files: a sub-directory that includes 4HHB_summary.json.
